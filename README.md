# PopMovies
Project with React, consuming the TMDB API using fetch.

### Demo

[https://pop-movies.vercel.app/](https://pop-movies.vercel.app/)

### Running it locally

### Core technologies

- React.js
- Styled Components
- Phosphoricons
- React Router
